If you want to upload a GeoFS add-on to geofs-addons, you must first fork this repository, 
then upload the required folder and .js file, and enter the name and creator name. 
The uploaded add-on undergoes AI verification, is approved, and is uploaded. 
If suspicious code is detected, your add-on will not be uploaded to geofs-addons.
